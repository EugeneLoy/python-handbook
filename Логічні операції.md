# Логічні операції

Логічні операції дозволяють об'єднувати кілька умов в один вираз та отримувати логічний результат (`True` або `False`). Вони допомагають створювати складніші умови для прийняття рішень у програмі, наприклад, перевірити, чи одночасно виконуються кілька умов або чи виконується хоча б одна з них.

## Логічне І (and)

Оператор `and` повертає `True` тільки тоді, коли обидві умови істинні. Якщо хоча б одна умова хибна, результат буде `False`.

```python
temperature = 25
time = 14

if temperature > 20 and time < 20:
    print("Можна йти на прогулянку")
else:
    print("Краще залишитися вдома")
```

Результат:
```
Можна йти на прогулянку
```

Якщо змінити значення так, що хоча б одна умова стане хибною:

```python
temperature = 15
time = 14

if temperature > 20 and time < 20:
    print("Можна йти на прогулянку")
else:
    print("Краще залишитися вдома")
```

Результат:
```
Краще залишитися вдома
```

## Логічне АБО (or)

Оператор `or` повертає `True`, якщо хоча б одна з умов істинна. Результат буде `False` тільки тоді, коли всі умови хибні.

```python
temperature = 5
wind_speed = 8

if temperature < 10 or wind_speed > 15:
    print("Потрібно одягнутися тепліше")
else:
    print("Можна одягнутися легше")
```

Результат:
```
Потрібно одягнутися тепліше
```

Якщо обидві умови хибні:

```python
temperature = 18
wind_speed = 8

if temperature < 10 or wind_speed > 15:
    print("Потрібно одягнутися тепліше")
else:
    print("Можна одягнутися легше")
```

Результат:
```
Можна одягнутися легше
```

## Логічне НІ (not)

Оператор `not` інвертує логічне значення: якщо умова істинна, `not` повертає `False`, якщо умова хибна — повертає `True`.

```python
is_raining = False

if not is_raining:
    print("Можна йти без парасольки")
else:
    print("Потрібна парасолька")
```

Результат:
```
Можна йти без парасольки
```

Якщо змінити значення:

```python
is_raining = True

if not is_raining:
    print("Можна йти без парасольки")
else:
    print("Потрібна парасолька")
```

Результат:
```
Потрібна парасолька
```

## Таблиця логічних операцій

| Оператор | Пояснення | Приклади |
|---|---|---|
| `a and b` | повертає `True`, якщо обидві умови істинні | `True and True # результат: True`<br>`True and False # результат: False`<br>`False and False # результат: False` |
| `a or b` | повертає `True`, якщо хоча б одна умова істинна | `True or False # результат: True`<br>`False or True # результат: True`<br>`False or False # результат: False` |
| `not a` | інвертує логічне значення | `not True # результат: False`<br>`not False # результат: True` |
