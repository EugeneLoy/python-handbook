# Змінні - присвоєння значень

Змінна — це ім'я, яке ми даємо значенню, щоб потім використовувати його в коді.

## Найпростіший приклад

```python
name = "Оля"
print(name)
```

Результат:
```
Оля
```

У цьому прикладі `name` — це змінна, якій присвоєно значення `"Оля"`. Знак `=` означає "присвоїти" або "зберегти в". Коли ми викликаємо `print(name)`, Python знаходить значення змінної і виводить його на екран.

## Присвоєння результату обчислення

Змінна може зберігати результат обчислення, наприклад, скільки яблук залишиться після частування друзів:

```python
apples = 8
friends = 3
apples_left = apples - friends
print(apples_left)
```

Результат:
```
5
```

## Навіщо використовувати змінні

Змінні роблять код зрозумілішим і простішим для зміни. Наприклад, якщо потрібно вивести одне і те саме значення кілька разів:

```python
book_price = 150
print("Книга коштує", book_price, "грн")
print("Зі знижкою 20%:", book_price * 0.8, "грн")
```

Результат:
```
Книга коштує 150 грн
Зі знижкою 20%: 120.0 грн
```

Якщо потрібно змінити ціну книги, достатньо змінити значення один раз у місці, де змінна створюється, і всі місця, де вона використовується, автоматично оновляться. Також надання імені значенню допомагає зрозуміти його роль у програмі: коли ми бачимо `book_price`, одразу зрозуміло, що це ціна книги, а не просто число 150.
