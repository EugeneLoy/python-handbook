# Використання None

`None` - це спеціальне значення в Python, яке означає "немає значення". Воно зручне, коли потрібно позначити відсутність результату.

Наприклад, знайдемо найдовше слово серед слів, які вводить користувач. Якщо список слів порожній, відсутність результату можна позначити за допомогою `None`:

```python
text = input("Введіть слова через пробіл: ")
words = text.split()

longest = None

for word in words:
    if longest is None or len(word) > len(longest):
        longest = word

if longest is None:
    print("Слова не введено.")
else:
    print("Найдовше слово:", longest)
```

Результат, якщо користувач введе декілька слів:

```text
Введіть слова через пробіл: кіт собака лисиця
Найдовше слово: лисиця
```

Результат, якщо користувач нічого не введе:

```text
Введіть слова через пробіл: 
Слова не введено.
```
