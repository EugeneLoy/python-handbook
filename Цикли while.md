# Цикли `while`

У деяких задачах наперед невідомо, скільки разів потрібно повторити дію. У таких випадках зручно використовувати цикл `while`: він виконує тіло циклу доти, доки умова залишається істинною.

Припустимо, маємо аркуш паперу товщиною `0.1` мм. Після кожного складання товщина подвоюється. Потрібно дізнатися, скільки разів треба скласти аркуш, щоб товщина стала не меншою за `1` мм.

У цій задачі наперед невідомо, скільки разів доведеться складати аркуш, тому зручно повторювати складання, поки товщина менша за `1` мм:

```python
thickness = 0.1
folds = 0

while thickness < 1:
    thickness = thickness * 2
    folds = folds + 1

print("Потрібно скласти (разів):", folds)
```

Результат:

```text
Потрібно скласти (разів): 4
```

У рядку:

```python
while thickness < 1:
```

перевіряється умова `thickness < 1`. Якщо вона істинна, тіло циклу виконується ще раз: товщина подвоюється, а кількість складань збільшується на `1`. Коли ж товщина стає не меншою за `1` мм, умова стає хибною і цикл зупиняється.
