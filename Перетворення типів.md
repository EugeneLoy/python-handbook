# Перетворення типів

Перетворення типів — це створення нового значення іншого типу на основі наявного значення. Перетворення необхідне тоді, коли операції вимагають певного типу: наприклад, числа використовуються для обчислень, а рядки — для побудови тексту.

Крім того, деякі оператори в Python працюють по-різному залежно від типу значень, з якими вони взаємодіють. Наприклад, оператор `+` для чисел виконує додавання, а для рядків — конкатенацію ("склеювання"). Якщо переплутати типи, результат може бути іншим, ніж очікується.

```python
a = "2"
b = "3"
print(a + b) # конкатенація рядків

x = 2
y = 3
print(x + y) # додавання чисел
```

Результат:

```text
23
5
```

У Python для перетворення типів використовують функції `int()`, `float()`, `str()` та інші:

* `int(value)` — повертає значення `value` як ціле число. Якщо `value` — це рядок, він повинен містити тільки цифри та, за потреби, знак (у іншому випадку виникне помилка). Якщо `value` — це дійсне число, дробова частина буде відкинута.

* `float(value)` — повертає значення `value` як дійсне число. Якщо `value` — це рядок, він має містити коректний запис дійсного або цілого числа.

* `str(value)` — повертає значення `value` як рядок (текст).

## Коли перетворення типів необхідне

**При перетворенні рядка з цифрами на число, для математичних обчислень**:

```python
pi_str = "3.141592653589793"
pi = float(pi_str)
result = 2 * pi
print("2 * pi =", result)
```

Результат:

```text
2 * pi = 6.283185307179586
```

У цьому прикладі спочатку значення числа пі записане як рядок (у змінній `pi_str`). Для математичних обчислень потрібно отримати саме число, тому використовується `float(pi_str)`, що дає змогу перетворити рядок у дійсне число. Якби ми намагались одразу виконати математичне обчислення з рядком (наприклад, `2 * pi_str`), Python видав би помилку, оскільки не можна множити рядок з текстом на число.

**При перетворенні числа на рядок, для побудови тексту повідомлення**:

```python
count = 5
message = "Кількість: " + str(count)
print(message)
```

Результат:

```text
Кількість: 5
```

У цьому прикладі ціле число `count` перетворюється на рядок за допомогою функції `str(count)`, щоб його можна було "склеїти" з іншим рядком. Така операція потрібна, оскільки у Python не можна напряму додавати рядок і число. Якщо пропустити перетворення і написати `"Кількість: " + count`, програма завершиться з помилкою.
