# Змінні - оновлення значень

Після того, як змінній присвоєно значення, це значення можна змінити. Оновлення змінної працює так само, як і присвоєння: використовуємо знак `=` і вказуємо нове значення.

## Найпростіший приклад

```python
price = 200
print("Ціна товару:", price, "грн")
price = 150
print("Ціна зі знижкою:", price, "грн")
```

Результат:
```
Ціна товару: 200 грн
Ціна зі знижкою: 150 грн
```

У цьому прикладі змінна `price` спочатку містить значення `200`, а потім його змінюємо на `150`. Коли ми присвоюємо нове значення змінній, яка вже існує, старе значення замінюється новим.

## Навіщо оновлювати значення змінних

Оновлення значень змінних дозволяє використовувати одну змінну для зберігання різних значень у різні моменти виконання програми. Це корисно, коли значення має змінюватися під час роботи програми.

Наприклад, якщо потрібно відстежувати баланс гаманця після покупок:

```python
balance = 100
print("Баланс до покупки:", balance, "грн")
balance = 65
print("Баланс після покупки:", balance, "грн")
```

Результат:
```
Баланс до покупки: 100 грн
Баланс після покупки: 65 грн
```
