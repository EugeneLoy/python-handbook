# Базові операції з числами

У Python цілі числа (значення типу `int`) та дійсні числа (значення типу `float`) можна використовувати у виразах, що дозволяє виконувати обчислення для розв’язування багатьох практичних задач.

```python
print(7 + 2)     # додавання 7 і 2
print(7 - 2)     # віднімання 2 від 7
print(7 * 2)     # множення 7 на 2
print(7 / 2)     # ділення 7 на 2
print(7 ** 2)    # піднесення 7 до ступеня 2
print(9 ** 0.5)  # квадратний корінь з 9
```

Результат:
```
9
5
14
3.5
```

## Оператор цілочисельного ділення `//` та залишок від ділення `%`

Оператор `//` ділить число націло, тобто повертає лише цілу частину результату. Оператор `%` повертає залишок (остачу) від ділення.

Ці оператори корисні, коли потрібно дізнатися, скільки разів одне число міститься в іншому (наприклад, скільки цілих порцій можна зробити із заданої кількості продукту), або визначити, чи ділиться число націло (наприклад, перевірити, чи число парне: якщо залишок від ділення на 2 дорівнює 0, число парне).


```python
print(9 // 4)   # цілочисельне ділення
print(9 % 4)    # залишок від ділення
```

Результат:
```
2
1
```

## Таблиця базових операторів з числами

| Вираз у Python | Пояснення | Приклади |
|---|---|---|
| `a + b` | додати `a` і `b` | `2 + 3 # результат: 5`<br>`1.5 + 0.5 # результат: 2.0` |
| `a - b` | відняти `b` від `a` | `7 - 4 # результат: 3`<br>`3.0 - 5 # результат: -2.0` |
| `a * b` | помножити `a` на `b` | `6 * 7 # результат: 42`<br>`2.5 * 2 # результат: 5.0` |
| `a / b` | поділити `a` на `b` | `7 / 2 # результат: 3.5`<br>`5 / 2 # результат: 2.5` |
| `a ** b` | піднести `a` до степеня `b` | `2 ** 3 # результат: 8`<br>`5 ** 2 # результат: 25` |
| `a ** 0.5` | квадратний корінь з `a` | `9 ** 0.5 # результат: 3.0`<br>`25 ** 0.5 # результат: 5.0` |
| `a // b` | цілочисельне ділення (ціла частина) | `5 // 2 # результат: 2`<br>`6 // 2 # результат: 3`<br>`8 // 3 # результат: 2` |
| `a % b` | остача від ділення | `5 % 2 # результат: 1`<br>`6 % 2 # результат: 0`<br>`8 % 3 # результат: 2` |
