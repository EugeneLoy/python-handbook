# Логічні вирази - операції порівняння

Операції порівняння дозволяють порівнювати значення та отримувати логічний результат (`True` або `False`). Вони допомагають перевіряти умови та приймати рішення в програмі, наприклад, чи достатньо грошей для покупки або чи пройшов учень іспит.

## Перевірка рівності (`==`)

Оператор `==` перевіряє, чи два значення рівні між собою:

```python
age = 18
print("Вік дорівнює 18:", age == 18)
print("Вік дорівнює 16:", age == 16)
```

Результат:
```
Вік дорівнює 18: True
Вік дорівнює 16: False
```

## Перевірка нерівності (`!=`)

Оператор `!=` перевіряє, чи два значення не рівні між собою:

```python
age = 18
print("Вік не дорівнює 18:", age != 18)
print("Вік не дорівнює 16:", age != 16)
```

Результат:
```
Вік не дорівнює 18: False
Вік не дорівнює 16: True
```

## Порівняння порядку (`>`, `<`, `>=`, `<=`)

Оператори порівняння порядку дозволяють перевіряти, чи одне значення більше або менше за інше:

- `>` — перевіряє, чи перше значення більше за друге
- `<` — перевіряє, чи перше значення менше за друге
- `>=` — перевіряє, чи перше значення більше або дорівнює другому
- `<=` — перевіряє, чи перше значення менше або дорівнює другому

```python
age = 18
print("Вік більше 17:", age > 17)
print("Вік менше 17:", age < 17)
print("Вік більше або дорівнює 18:", age >= 18)
print("Вік менше або дорівнює 16:", age <= 16)
```

Результат:
```
Вік більше 17: True
Вік менше 17: False
Вік більше або дорівнює 18: True
Вік менше або дорівнює 16: False
```

## Ланцюгові порівняння

Python дозволяє об'єднувати кілька порівнянь в один вираз. Наприклад, можна перевірити, чи значення знаходиться в певному діапазоні:

```python
age = 18
print("Вік від 13 до 17:", 13 <= age <= 17)
print("Вік від 18 до 65:", 18 <= age <= 65)
```

Результат:
```
Вік від 13 до 17: False
Вік від 18 до 65: True
```

| Оператор | Пояснення | Приклади |
|---|---|---|
| `a == b` | перевірити, чи `a` дорівнює `b` | `5 == 5 # результат: True`<br>`5 == 3 # результат: False` |
| `a != b` | перевірити, чи `a` не дорівнює `b` | `5 != 3 # результат: True`<br>`5 != 5 # результат: False` |
| `a > b` | перевірити, чи `a` більше за `b` | `5 > 3 # результат: True`<br>`3 > 5 # результат: False` |
| `a < b` | перевірити, чи `a` менше за `b` | `3 < 5 # результат: True`<br>`5 < 3 # результат: False` |
| `a >= b` | перевірити, чи `a` більше або дорівнює `b` | `5 >= 5 # результат: True`<br>`3 >= 5 # результат: False` |
| `a <= b` | перевірити, чи `a` менше або дорівнює `b` | `5 <= 5 # результат: True`<br>`5 <= 3 # результат: False` |
| `a < b < c` | перевірити, чи `b` знаходиться між `a` і `c` | `20 < 22 < 25 # результат: True`<br>`15 < 22 < 20 # результат: False` |
