# Виклик функцій з використанням ключових аргументів

Коли ми викликаємо функцію, ми можемо передавати значення не тільки за позицією, але й за назвою аргумента. Такі аргументи називаються ключовими аргументами. Вони дозволяють явно вказати, якому аргументу функції належить кожне значення, що робить код більш читабельним і зручним.

## Аргумент sep функції print

Функція `print` має аргумент `sep`, який визначає, який символ буде використовуватися для розділення кількох значень. За замовчуванням між значеннями вставляється пробіл, але ми можемо змінити це.

Наприклад, можна використати `sep` для форматування часу доби:

```python
hour = 14
minute = 30
second = 45
print(hour, minute, second, sep=":")
```

Результат:
```
14:30:45
```

У цьому прикладі ми передаємо три числа (годину, хвилину та секунду) і вказуємо, що між ними має бути двокрапка замість пробілу. Це дозволяє отримати зручний формат часу.

## Аргумент end функції print

Аргумент `end` визначає, що буде надруковано після всіх значень. За замовчуванням після кожного виклику `print` додається спеціальний символ нового рядка (в Python він записується як `"\n"`), тому наступний виклик `print` друкує на новому рядку. Але ми можемо змінити це.

Наприклад, можна використати `end` для друкування кількох повідомлень в одному рядку:

```python
print("Завантаження", end="...")

# після завантаження повідомимо про заверщення

print("Готово!")
```

Результат:
```
Завантаження...Готово!
```

Без використання `end` ці два виклики `print` надрукували б повідомлення на різних рядках. Але вказавши `end="..."`, ми змінили поведінку: після першого повідомлення додається три крапки замість переходу на новий рядок, тому друге повідомлення друкується одразу після них.
