# Базові операції з рядками

Рядки у Python — це послідовності символів, з якими часто працюють під час друкування повідомлень або роботи з текстом.

Усі рядки в Python це значення типу `str`.

## Об'єднання рядків оператором `+` (конкатенація)

Конкатенація — це об'єднання кількох рядків в один. Використовується, коли потрібно зібрати текст із частин.

```python
print("Привіт, " + "світе!")
```
Результат:
```
Привіт, світе!
```

## Повторення рядка оператором `*`

Оператор `*` дозволяє повторити рядок задану кількість разів. Це зручно для простого форматування під час друкування, створення розділювачів, повторення шаблонів.

```python
print("-" * 11)
print(("Ха! " ) * 3)
```

Результат:
```
-----------
Ха! Ха! Ха! 
```
