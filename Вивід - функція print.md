# Вивід - функція print

Функція `print` друкує повідомлення на екрані (у консолі). Це найперша команда, з якої зручно починати знайомство з Python.

## Найпростіший приклад
```python
print("Привіт, світе!")
```
Результат:
```
Привіт, світе!
```

У цьому прикладі функція `print` отримує одне значення — рядок `"Привіт, світе!"`. Коли Python виконує цю команду, він друкує (виводить) усе, що знаходиться в лапках, на екран.

`print` — це вбудована функція у Python, яка завжди пишеться малими літерами. Коли ми її викликаємо, усередині дужок ми вказуємо, що саме хочемо надрукувати: текст, числа, або навіть результат обчислень.

## Вивід чисел

Функція `print` може друкувати не тільки рядки, але й числа або інші значення. Наприклад, можна вивести ціле або дійсне число:

```python
print(2024)
print(3.14)
```
Результат:
```
2024
3.14
```

## Вивід декількох значень за раз
Можна передати кілька значень через кому — `print` автоматично вставить між ними пробіли:
```python
print("Ім'я:", "Оля", "Вік:", 14)
```
Результат:
```
Ім'я: Оля Вік: 14
```

## Вивід результатів обчислень
За допомогою `print` можна друкувати як значення, так і результати обчислення виразів (наприклад, додавання):
```python
print("Сума:", 2 + 3)
```
Результат:
```
Сума: 5
```
