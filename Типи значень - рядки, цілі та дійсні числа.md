# Типи значень - рядки, цілі та дійсні числа

Коли ми щось виводимо через `print`, Python працює з різними видами даних. Ці види називаються типами. Типи існують для того, щоб комп’ютер знав, як поводитися зі значенням: як його зберігати, які операції дозволені і як показувати результат.

## Рядок (`str`)
Рядок — це текст у лапках. Лапки можуть бути одинарні `'...'` або подвійні `"..."`.

```python
print("Привіт")
print('Python — це веселo!')
```

Результат:
```
Привіт
Python — це веселo!
```

Рядки зручно використовувати для імен або назв предметів, повідомлень або будь-якого іншого тексту. З рядками працюють текстові операції, наприклад «склеювання» (конкатенація):

```python
print("Привіт, " + "світе!")
```

Результат:
```
Привіт, світе!
```

## Ціле число (`int`)
Це числа без дробової частини: ..., -2, -1, 0, 1, 2, ...

```python
print(10)
print(2 + 3)
```

Результат:
```
10
5
```

Цілі числа підходять для підрахунків кількості предметів або інших цілих величин. До них застосовуються арифметичні операції: додавання, віднімання, множення.

## Дійсне число (`float`)
Це число з дробовою частиною. У Python використовується крапка як роздільник дробової частини.

```python
print(3.14)
print(0.5 + 0.25)
```

Результат:
```
3.14
0.75
```

`float` зручно використовувати для вимірювань, відсотків, значень з комою.

## Порівняльна таблиця основних типів

У цій таблиці наведено найпоширеніші типи даних, які використовуються у Python для роботи з текстом та числами.

| Тип у Python | Назва                | Приклади значень у Python    |
|--------------|----------------------|------------------------------|
| `str`        | Рядок (string)       | `"Привіт"`, `"Python"`, `""` |
