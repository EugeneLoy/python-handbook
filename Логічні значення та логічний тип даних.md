# Логічні значення та логічний тип даних

Логічний тип даних (`bool`) — це тип, який може зберігати лише два значення: `True` (істина) або `False` (хибність). 

Логічні значення зручно використовувати для позначення станів "так" або "ні", або для прийняття рішень.

```python
print(True)    # надрукує значення True (істина)
print(False)   # надрукує значення False (хибність)
```

Результат:
```
True
False
```

## Використання зі змінними

Логічні значення можна зберігати у змінних, щоб відстежувати різні стани в програмі. Наприклад, чи увімкнено світло в кімнаті:

```python
light_on = False
print("Світло увімкнено (так - True, ні - False):", light_on)
light_on = True # увімкнути світло
print("Світло увімкнено (так - True, ні - False):", light_on)
```

Результат:
```
Світло увімкнено (так - True, ні - False): False
Світло увімкнено (так - True, ні - False): True
```

Замість використання рядків `"так"` або `"ні"`, або чисел `1` та `0`, логічні значення роблять код більш зрозумілим і допомагають уникнути помилок. Коли ми бачимо змінну з логічним значенням, одразу зрозуміло, що вона описує якийсь стан, який може бути лише істинним або хибним.
