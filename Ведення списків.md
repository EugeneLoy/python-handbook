# Ведення списків

Часто потрібно зчитати не одне значення, а одразу багато - наприклад, список слів<!-- або список чисел-->.

## Зчитування списку слів за допомогою `split()`

Метод рядка `split()` розбиває рядок на частини за пробілами і повертає список рядків.

```python
print("Введіть слова через пробіл:")
words = input().split()

print("Список слів:", words)
```

Результат:
```
Введіть слова через пробіл:
яблуко груша банан
Список слів: ['яблуко', 'груша', 'банан']
```

## Зчитування списку цілих чисел

Метод `split()` завжди повертає список рядків, навіть якщо ми вводимо цифри. 
Якщо потрібно зчитати список цілих чисел, треба перетворити кожен рфдок в списку на ціле число за допомогою функціх `int()`.

Найчастіше для цього використовують функцію `map()`, яка застосовує вказану функцію до всіх елементів послідовності.

```python
print("Введіть цілі числа через пробіл:")
numbers = list(map(int, input().split()))

print("Список чисел:", numbers)
```

Результат:
```
Введіть цілі числа через пробіл:
10 3 7 25
Список чисел: [10, 3, 7, 25]
```

Таким чином, `split()` допомагає перетворити один рядок введених даних на список рядків, а `map(int, ...)` - перетворити всі елементи цього списку на цілі числа. -->
